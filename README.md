# Configuration files for `Resil Networks` 

Each network must have branch with `spec.json`, `bootnodes.txt`.
If there are scripts, they should be in `script/` directory.

## Network IDs

- 172 or `0xAD` - Testnet
- XX or `0xXX` - Mainnet, pending
